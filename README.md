# Crop Recommendation Using Machine Learning for Sustainable Agriculture

## Project Overview

This project develops a machine learning-based crop recommendation system using soil nutrient and environmental data. The aim is to predict the most suitable crop based on input features such as nitrogen, phosphorus, potassium, temperature, humidity, pH and rainfall.

The project follows a complete data science workflow including exploratory data analysis, preprocessing, model training, model evaluation, hyperparameter tuning and final crop prediction testing.

## Dataset

The dataset used in this project is the **Crop Recommendation Dataset** from Kaggle.

Dataset link:  
https://www.kaggle.com/datasets/atharvaingle/crop-recommendation-dataset

The dataset contains:

- 2,200 records
- 8 columns
- 7 input features
- 1 target variable
- 22 crop classes

## Project Workflow

The project follows these main steps:

1. Dataset loading
2. Exploratory Data Analysis
3. Missing value and duplicate checking
4. Feature and target separation
5. Target label encoding
6. Train-test split
7. Feature scaling
8. Model training
9. Model evaluation
10. Hyperparameter tuning
11. Feature importance analysis
12. Manual crop prediction testing

## Machine Learning Models Used

The following models were trained and compared:

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- K-Nearest Neighbours
- Support Vector Machine
- Gradient Boosting Classifier
